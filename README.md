# 🌿 Natural Farm App — Payment Integration

Natural Farm App is a digital platform that allows users to purchase fresh, organic farm products directly from local farmers.

This project demonstrates how secure online payments can be integrated into the platform using Quickteller (Interswitch Webpay).

---

## 🌱 About the Project

The goal of Natural Farm App is to:

- Connect customers with local farmers
- Promote healthy and organic food consumption
- Provide a simple and secure way to make payments online

---

## 💳 Payment Integration Features

- Secure checkout page
- Payment method selection (Card / Bank Transfer)
- Integration with Quickteller Webpay
- Automatic redirection to payment gateway
- Demo success page for presentation

## ⚙️ How It Works

1. User selects farm products
2. Proceeds to checkout
3. Chooses payment method
4. Backend generates secure transaction
5. Redirect to Quickteller payment page

---

## 🧪 Demo Note

This project uses Quickteller sandbox environment.  
Transactions may not complete due to merchant activation limits.

A demo success page is included for presentation.
